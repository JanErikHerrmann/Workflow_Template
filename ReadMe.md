The purpose of these tests is to test the capabilities from an AI and human developers.

When using AI:
- Please use AI as much as possible to complete the tasks. If the AI can´t handle the task or needs some help, you can step in and assist the AI.
- You can use any modules/plugins you like as long as Copilot requires them to be able to solve the task.
- This test is to primarily evaluate the GenAI and not the developer's knowledge of a specific framework. That means that AI should be used wherever possible to complete the tasks.

When not using AI:
- You are not allowed to use the GitHub copilot AI within the IDE or any other AI.
- You can use any modules/plugins you like as long as they are free to use.
- This test is to primarily evaluate the developer's knowledge and skill of web development. That means that AI should not be used to complete the tasks.

Please ensure that when starting the development you make an empty commit to signal the start time.
When done, make another empty commit to signal the end time. If possible, please make a commit after each completed task. (This is not mandatory but helps track progress and time taken for each task.)

Each task should be completed using best practices in web development, ensuring responsiveness and accessibility.
The maximum time allocated for all tests in this project is 1 hour, which is intentionally challenging.
